# Sistema de Gerenciamento de Contas Bancarias
Trabalho final da cadeira de POO do professor Ênio.

>O Sistema consiste em um sistemas de gerenciamento de contas bancárias, incluindo interações com o cliente por interface gráfica (Java Swing).

>O objetivo real do projeto é trabalhar todos os conceitos da Programação Orientada á Objetos (POO).

Principais telas e funcionalidades:
-
> **Tela inicial**: é a tela principal do programa, a qual pode-se executar o cadastro dos clientes e a criação de contas de determinado cliente. Além da apresentação do nome do Sistema.

>**Tela de operações bancárias**: nesta tela pode ser feito as seguintes operações bancárias entre as contas cadastradas: saque, depósito, transferência.  

>**Persistência**: foi feita em arquivo *.txt*, o qual funciona como o banco de dados do sistema, é resgatado dados de dentro do arquivo de texto e é escrito dados dentro do mesmo.

O que foi utilizado:
-
* Java puro
* Java swing 
* Git (para o versionamento)
